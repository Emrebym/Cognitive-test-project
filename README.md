# 🧠 Cognitive Test Project

A React-based cognitive assessment application with bilingual EN/TR support, featuring multiple question types for measuring cognitive abilities and IQ scoring.

## Features

- **Multiple test types** — n-back, spatial grid, story recall, pattern recognition
- **Bilingual support** — full English and Turkish language toggle
- **IQ scoring** — calculated score based on performance
- **Pause & resume** — take breaks during the assessment
- **Progress tracking** — visual progress bar throughout the test
- **Responsive design** — works on desktop and mobile
